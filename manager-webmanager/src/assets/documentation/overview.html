DEF Web-Manager

Der DEF Web-Manager wurde in erster Linie für das Monitoring entwickelt und umfasst die folgenden drei Menü-Punkte:

  • Programme
  • Ressourcen
  • Bibliothek

Im Folgenden wird für jeden dieser Punkte eine kurze Übersicht gegegeben, welche Funktionalität der DEF Web-Manager bietet. Die genaue Dokumentation inklusive der Anleitung für die richtige Anwendung kann über die Navigationsleiste am linken Browser-Rand erreicht werden. Durch Klicken auf den Titel "DEF Web-Manager Dokumentation" am oberen Rand des Browsers kann auf diese Seite zurückgekehrt werden.


Programme

Der DEF Web-Manager dient für das Monitoring der laufenden und abgeschlossenen Programme, die im DEF bearbeitet werden bzw. wurden. Es wird eine Übersicht über alle Programme geboten, unterteilt in laufende und abgeschlossene Programme, sowie eine Detail-Ansicht der einzelnen Programme, Jobs und Tasks.

Zusätzlich zu deren Überwachung wird die Möglichkeit geboten, laufende Programme, Jobs und Tasks abzubrechen. Weiters können abgeschlossene Programme über den DEF Web-Manager gelöscht werden.

Die umfangreiche Dokumentation zum Programme-Menüpunkt im DEF Web-Manager kann über den Punkt "Programme" in der Naviationgsleiste am linken Browser-Rand erreicht werden.


Ressourcen

Neben dem Monitoring der Programme ist der DEF Web-Manager ebenfalls für das Monitoring der Ressourcen gedacht. So werden in einer Cluster-Übersicht alle Cluster dargestellt, die das DEF aktuell für Berechnungen verwenden kann, inklusive der direkten Anzeige der aktuellen Auslastung des gesamten Clusters.

Sowohl für die einzelnen Cluster als auch für die Worker, die sich innerhalb der Cluster befinden, wird eine Detail-Ansicht mit allen relevanten Informationen zur jeweiligen Komponente inklusive der aktuellen Auslastung zur Verfügung gestellt.

Aktuell kann der DEF Web-Manager nur für das Monitoring der Ressourcen verwendet werden, in Zukunft soll jedoch auch die Möglichkeit geboten werden, über den DEF Web-Manager vorhandene Cluster bzw. Worker aus dem System zu entfernen und neue hinzuzufügen.

Die umfangreiche Dokumentation zum Ressourcen-Menüpunkt im DEF Web-Manager kann über den Punkt "Ressourcen" in der Navigationsleiste am linken Browser-Rand erreicht werden.


Bibliothek

Der Menüpunkt "Bibliothek" im DEF Web-Manager befindet sich aktuell noch in Bearbeitung, darum ist hierfür auch noch keine Dokumentation vorhanden. Es ist jedoch angedacht, dass über den DEF Web-Manager alle in der Bibliothek vorhandenen Routinen dargestellt und durchsucht, sowie neue Routinen erzeugt und hochgeladen werden können.

Die umfangreiche Dokumentation zum Bibliothek-Menüpunkt im DEF Web-Manager wird über den Punkt "Bibliothek" in der Navigationsleiste am linken Browser-Rand erreicht werden können.
